Bienvenidos a Francoz APIs

Endpoints disponibles:

    Usuarios

  • /usuarios : [GET] Obtener la lista de usuarios
  • /usuarios/registrar : [POST] Simula la creación de un usuario
  • /auth/login : [POST] Simula la verificación de credenciales
  • /usuarios/editar : [PUT] Simula la edicion de datos de un usuario


    • Categorias

  • /categorias : [GET] Obtener la lista de categorías


    • Roles

  • /roles : [GET] Obtener la lista de roles


    • Productos

  • /productos : [GET] Obtener la lista de productos


    • Clientes

  • /clientes : [GET] Obtener la lista de clientes


    • Proveedores

  • /proveedores : [GET] Obtener la lista de proveedores


    • Ventas

  • /ventas : [GET] Obtener la lista de ventas
  • /ventas?idusuario=5 : [GET] Obtener la lista de ventas para el IdUsuario = 5


    • Compras

  • /compras : [GET] Obtener la lista de compras
  • /compras?idusuario=3 : [GET] Obtener la lista de compras para el IdUsuario = 3


    • Permisos

  • /permisos?idusuario=1 : [GET] Obtener la lista de permisos para el IdUsuario = 1


    • DetalleCompras

  • /detallescompra?idCompra=1 : [GET] Obtener la lista de DetalleCompras para el IdCompra = 1

    • DetalleVentas

  • /detallesventa?idVenta=1 : [GET] Obtener la lista de DetalleVentas para el IdVenta = 1
