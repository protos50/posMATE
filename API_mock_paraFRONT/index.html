Bienvenidos a Francoz APIs

Endpoints disponibles:

  • /usuarios : [GET] Obtener la lista de usuarios
  • /usuarios/registrar : [POST] Simula la creación de un usuario
  • /login : [POST] Simula la verificación de credenciales
  • /usuarios/editar : [PUT] Simula la edicion de datos de un usuario
  • /categorias : [GET] Obtener la lista de categorías
